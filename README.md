# sales-performance-Power-BI
<>Project Overview
This project is a Sales Performance Dashboard created using Power BI.
It helps analyze sales data using KPIs, charts, and filters.

## Tools Used
- Power BI Desktop
- DAX (Basic)
- Excel (Data Source)

## Key Features
- Total Sales, Total Quantity, Average Sales
- Sales by Region
- Sales by Product
- Monthly Sales Trend
- Interactive slicers for filtering

## Files Included
- Sales_Performance_Dashboard.pbix – Power BI report file
- dashboard_preview.png – Dashboard screenshot

## Learning Outcome
- Data transformation using Power Query
- Creating measures using DAX
- Designing interactive dashboards
